Welkom bij CSS Selectors

Maak het ul.menu op als een top menu. Waarbij alle eerste elementen onder de ul naast elkaar zweven.
Zorg dat het er uit ziet als blokken naast elkaar, toon de lijst onder articles alleen als de muis eroverheen gaat(:hover).

De HTML mag niet worden aangepast.


Maak gebruik van minimaal de volgende selectors:

  • ul.menu > li
  • ul.menu > li > ul
  • ul.menu li
  • ul.menu li:hover
  • ul.menu > li:last-child:hover
  • ul.menu + div + div
